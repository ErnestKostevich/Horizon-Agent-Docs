# Support

## Self-help first

1. Check the [FAQ](faq.md) — most common questions answered there
2. Run `horizon doctor` — auto-diagnoses configuration issues, can
   often auto-fix with `horizon doctor --fix`
3. Search the [GitHub issues](https://github.com/ErnestKostevich/horizon-genesis/issues)
   — someone may have hit your problem already

## Reporting a bug

File a new issue at
[github.com/ErnestKostevich/horizon-genesis/issues](https://github.com/ErnestKostevich/horizon-genesis/issues).

Please include:

1. Your OS + version
2. Horizon version (`horizon version`)
3. Steps to reproduce
4. Expected vs actual behaviour
5. Any error messages (run with `--verbose`)
6. If applicable, the output of `horizon doctor --json`

**Do not** include your API keys or anything that decrypts to them.

## Feature requests

Same place — open an issue with the "feature request" label. We
prioritise based on user demand, technical fit, and impact.

## Security disclosure

For security issues — **do not** open a public issue.

Email **ernest2011kostevich@gmail.com** directly with:
- A description of the issue
- Reproduction steps
- The version affected

You'll get a response within 48 hours and credit in the security
acknowledgements once the fix ships.
